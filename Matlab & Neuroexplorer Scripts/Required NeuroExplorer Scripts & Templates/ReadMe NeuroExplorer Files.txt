The .nsc files in the Scripts folder must be copied to NeuroExplorer's Scripts folder.
The .ntp files in the Templates folder must be copied to NeuroExplorer's Templates folder.
The calcWFcorrelationMLTrigd.m file should be added to a folder inside your Matlab installation's Set Path. 

To find which folders NeuroExplorer uses: 
run Neuroexplorer on the main menu bar select Scripts > Script options
or Templates > Templates options to see folder used by Nex
